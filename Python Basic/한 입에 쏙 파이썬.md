# 파트4. Chapter 3. 재료 모으기

- 리스트 List 
  - 선언형태 : [값1, 값2, 값3, ... ]
  - 리스트 내부에는 어떤 자료형도 입력 가능
  - 중첩 구조 또한 가능
  - 값 추가하기
    - append(값) : 리스트 맨 뒤에 추가
  - 값 제거하기
    - del 리스트[인덱스] : 리스트의 인덱스에 해당하는 값 제거
    - del은 키워드 Keyword
  - 여러 값을 가져오기 (Indexing)
    - [start : end]  : start 부터 end 전까지
    - 기존 데이터에 영향 주지 않음
